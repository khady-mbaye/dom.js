DOM EN JAVASCRIPT

Definition

DOM(Document Object Model) est une interface de programmation d'application (API) permettant de manipuler des documents HTML.

Le DOM représente un document HTML sous la forme d'une arborescence de nœuds. Le DOM fournit des fonctions qui vous permettent d'ajouter, de supprimer et de modifier efficacement des parties du document.
